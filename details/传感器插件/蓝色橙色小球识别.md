## 4. 识别算法说明（OpenCV）

### 4.1 处理流程
1. ROS 图像转 OpenCV（`cv_bridge`）
2. BGR -> HSV
3. 按颜色阈值二值化
4. 形态学开/闭操作去噪
5. 轮廓提取
6. 用面积、圆度、半径筛选“球形目标”
7. 在图像上绘制圆框与类别文本
8. 发布检测结果字符串 + 调试图像

### 4.2 当前阈值（可后续微调）
- 蓝色：`H[95,135], S[80,255], V[60,255]`
- 橙色：`H[5,25], S[120,255], V[80,255]`

### 4.3 几何筛选条件
- 最小面积：`area >= 120`
- 最小半径：`radius >= 6`
- 圆度：`4πA / P² >= 0.60`

说明：以上条件用于抑制噪声块和非球形高亮区域。

## 8. 常见问题与排查

### 8.1 有窗口但识别不到球
优先排查：
1. 小球颜色是否落在 HSV 阈值范围内
2. 光照是否导致颜色漂移（V/S 变化大）
3. 球在画面中是否太小（半径阈值过高）

### 8.2 原图有，标注图无
检查识别节点是否在运行：
```bash
ros2 node list | grep ball_detector
ros2 topic hz /ball/debug_image
```

### 8.3 话题有但显示空白
检查 QoS 与话题是否一致：
```bash
ros2 topic info /rgb/image_raw -v
ros2 topic info /ball/debug_image -v
```
